# Git
## Linux
### Instalación
1. Actualización del listado de repositorios e instalación de Git:
sudo apt update
sudo apt install git
2. Comprobar instalación:
git --version
### Configuración inicial
1. Configurar identidad de Git:
git config --global user.name "Nombre de Usuario"
git config --global user.email "email@ejemplo.com"
### Iniciar proyecto
1. Navegar al directorio que se quiere subir a GitHub.
2. Inicializar Git en la carpeta:
git init
3. Comprobar el estado:
git status
4. Agregar los archivos:
	a) Agregar todos los archivos:
	git add .
	b) Agregar archivos concretos:
	git add "nombre_de_archivo"
5. Crear un "commit" (estado actual de los archivos):
git commit -m "nombre commit"
6. Conectar el repositorio local con el remoto:
	- Copiar la URL del repositiorio vacío de GitHub:
		- Por ejemplo:
		git remote add origin https://github.com/Josean-mr/JavaScript.git
7. Subir (push) los archivos a GitHub:
	a) git push -u origin main
	- Si es la primera vez pedirá una contraseña. Debemos crear un token.
		1. En nuestro perfil vamos a "Settings" - > "Developer settings".
		2. Creamos un "Token (classic)" marcando las opciones de repositorios.,
	b) Escribir nombre de usuario.
	c) Pegar el token.
### Actualizar proyecto
1. Agregar nuevos archivos.
	a) Todos los archivos:
	git add .
	b) Archivos concretos:
	git add "nombre_archivo"
2. Crear un "commit":
 git commit -m "nombre commit"
3. Cargar los archivos:
git push
