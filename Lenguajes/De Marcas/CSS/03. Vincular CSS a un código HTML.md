Dos formas:
- a) CSS en línea.
- b) CSS externo.
<h2>a) CSS en línea</h2>
Se trata de aplicar los estilos directamente en las etiquetas HTML. Cada vez se utiliza menos.
```html
<p style="background:red;">El fondo del párrafo es rojo</p>
```
<h2>b) CSS como documento externo</h2>
La mejor opción es crear un documento CSS externo y vincularlo al documento HTML.
```html
<!DOCTYPE html>
<html lang="es">

<head>
	<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width, initial scale=1.0">
	<link href="styles.css" rel="StyleSheet" type="text/css">
	<title>Insertar CSS</title>
</head>

<body>

</body>
```
Importar hojas de estilo:
Las hojas de estilo tambien se pueden importar con <font style="color:#08f; font-weight:bold">@import</font>.
<div class="ejemplo">Ejemplo:</div>
```html
@import '/css/estilo1.css';
@import "/css/estilo1.css";
@import url('/css/estilo1.css');
@import url("/css/estilo1s.css");
```
