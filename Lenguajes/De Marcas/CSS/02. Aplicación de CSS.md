# Uso de selectores
Diferentes opciones basadas en selectores de:
-  a) Etiquetas.
- b) Clases.
- c) Identificadores
## a) Selectores basados en etiquetas
Cada etiqueta HTML puede tener declarada un estilo.
Ejemplo:
```CSS
p {
	color:green;
	text-align:center;
}
```
En ocasiones podemos agrupar propiedades comunes.
Ejemplo:
```CSS
	h1, h2, h3 }
	color: #8FAA28;
	font-weight: normal;
}

h1 {
	font-size: 4em;
}

h2 {
	font-size: 2em;
}

h3 {
	font-size: 2em;
}
```
## b) Selectores basados en clases
Se suelen utilizar estos selectores para aplicar el mismo estilos en diferentes elementos. Existen diferentes métodos de aplicar estilos:
1. A los contenedores (div).
2. A las etiquetas.
A las etiquetas. Podemos aplicar un filtro para que solamente se aplique el estilo en los elementos que se establezcan:
```CSS
h1.color_gris {
	color: grey
	}
```
En HTML sería:
```html
<h1 class="color_gris">Texto de color gris</h1>
```
## c) Selectores basados en identificadores
Se suelen utilizar para aplicar estilos CSS a un sólo elemento de la página.
También es posible combinar el selector id con otros selectores para restringir su alcance.
```CSS
p#advertencia {
	color:blue;
}
