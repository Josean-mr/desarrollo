- Herramienta de desarrollo y administración de bases de datos diseñada para facilitar las tareas relacionadas con bases de datos Oracle.
- Proporciona:
	- [Oracle SQL Developer](https://www.filehorse.com/es/download-oracle-sql-developer-64/download/):
		- Entorno gráfico integral que permite a los desarrolladores, DBAs (Administradores de Bases de Datos) y analistas interactuar de manera efectiva las bases de datos Oracle.
	- [PL/SQL Developer](https://pl-sql-developer.software.informer.com/11.0/):
		- Plataforma dedicada para la creación y depuración de procedimientos almacenados, funciones y desencadenadores en Oracle.
	- [DBeaver](https://dbeaver.io/download/):
		- Herramienta de administración de bases de datos de código abierto que proporciona soporte para Oracle, entre otros sistemas de gestión de bases de datos.
		- Destaca por admitir múltiples sistemas de basses de datos en una sola interfaz.
