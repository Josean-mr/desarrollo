# Numeric
- Número en coma flotante desempaquetado
- <font style="color:green; font-weight:bold">Ejemplo:</font>
```sql
NUMERIC(16,2)
```
# CHAR
- Se utiliza para almacenar cadenas de longitud fija con rango 1-255 caracteres.
- <font style="color:green; font-weight:bold">Ejemplo:</font>
```sql
CHAR(20)
```
# VARCHAR
- Se utiliza para almacenar cadenas con longitud variable con rango 1-255 caracteres.
- <font style="color:green; font-weight:bold">Ejemplo:</font>
```sql
VARCHAR(50)
```
# DATE
- Datos de tipo fecha.
- Formato por defecto YYYY MM DD.
# DATETIME
- Combina fecha y hora.
- Formato de almacenamiento año-mes-día horas:minutos:segundos.
