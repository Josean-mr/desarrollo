# ¿Qué es un evento?
- Es una acción que ocurre en un momento específico, ya sea en una fecha y hora programada o tras una acción específica sobre la base de datos.
- Algunos ejemplos de eventos programables incluyen la ejecución de copias de seguridad programadas, la actualización de estadísticas de la base de datos, la ejecución de tareas de mantenimiento, etc.
- El <font style="color:green; font-weight:bold">ejemplo</font> más tipo de evento en una base de datos es el <font style="color:#08F; font-weight:bold">trigger</font> o disparador. Un trigger es un objeto que se asocia a una tabla y se ejecuta automáticamente cuando ocurre un evento específico en dicha tabla.
<font style="color:salmon; font-weight:bold">Estructura básica de un trigger:</font>
```sql
CREATE OR REPLACE TRIGGER nombre_trigger
[BEFORE | AFTER]
[INSERT | DELETE | UPDATE {OF columns} ON nombre_tabla]
	[REFERENCING OLD AS old NEW AS new]
	[FOR EACH ROW]
	[WHEN (condicion)]
	[DECLARE]
	-- Declaraciones variables locales
	BEGIN
	-- Lógica del trigger
	-- Sentencias SQL y/o llamadas a procedimientos almacenados
	[EXCEPTION]
	-- Manejo de excepciones
END ;
```
- Las cláusulas <font style="color:#08F; font-weight:bold">BEFORE</font> y <font style="color:#08F; font-weight:bold">AFTER</font> hacen referencia al momento en el que debe ejecutarse el disparador.
	- <font style="color:#08F; font-weight:bold">BEFORE</font> indica que el trigger debe ejecutarse antes de realizar la operación sobre la tabla.
	- <font style="color:#08F; font-weight:bold">AFTER</font> indica que el trigger se ejecutará después de realizar la operación pertinente sobre la tabla.
- Las cláusulas <font style="color:#08F; font-weight:bold">INSERT</FONT>, <font style="color:#08F; font-weight:bold">UPDATE</font> y <font style="color:#08F; font-weight:bold">DELETE</font> hacen referencia a la operación que desencadenará la ejecución del trigger.
## Operadores <font style="color:#08F">OLD</font> y <font style="color:#08F">NEW</font>
- El operador <font style="color:#08F; font-weight:bold">OLD</font> hace referencia a los valores antes de la operación.
- El operador <font style="color:#08F; font-weight:bold">NEW</font> hace referencia a los valores después de la operación.
