# Funciones y funciones definidas por el usuario.
-  Existen **dos tipos de funciones**: las <font style="color:#08F; font-weight:bold">funciones incorporadas</font> o predefinidas por el sistema y las <font style="color:#08F; font-weight:bold">funciones definidas por el usuario</font>.
- La principal diferencia entre las funciones y los procedimientos, es que las funciones siempre devuelven un valor, para la cual se utiliza la cláusula RETURN.
<font style="color:salmon; font-weight:bold">Sintaxis:</font>
```SQL
CREATE [OR REPLACE] FUNCTION <func_name> [(<param1> [IN|OUT|IN OUT])] <type>, <param2> [IN|OUT|IN OUT] <type>,...)]
RETURN tipo_dato_retorno
AS -- declaración de variables locales
BEGIN - Sentencias
RETURN VALOR
[EXCEPTION] -- Sentencias control de excepción
END [<func_name>];
```
## <font style="color:#08F; font-weight:bold">Funciones incorporadas:</font>
- Son las proporcionadas por Oracle como parte del conjunto estándar de funciones de la base de datos.
- <font style="color:green; font-weight:bold">Ejemplos:</font>
	- SUM
	- AVG
	- MAX
	- MIN
	- TO_CHAR
	- UPPER
	- LOWER
	- ...
## <font style="color:#08F; font-weight:bold">Funciones definidas por el usuario:</font>
- Son funciones personalizadas creadas por los usuarios.
- Tienen una estructura similar a los procedimientos almacenados.
## Ejecutar una función
### Existen dos opciones:
- Llamarla desde una consulta SQL:
```SQL
SELECT <func_name> ([param]) FROM DUAL
```
- Crear un bloque PL/SQL:
```SQL
DECLARE
	v_total <type>;
BEGIN
	v_total := <func_name>([param]);
	dbms_output.put_line('El resultado es: ' || v_total);
end ;
```
### <font style="color:green; font-weight:bold">Ejemplos:</font>
**Parámetro incremento.**
Dado el precio de un artículo pasado como parámetro, devuelva el precio incrementado en 10.
```sql
CREATE OR REPLACE FUNCTION incremento (valor IN NUMBER)
	RETURN NUMBER IS
	BEGIN
		RETURN valor + 10;
	END;
```
#### Dos opciones
##### Opción 1:
 -  Llamarla desde una consulta SQL
```sql
SELECT incremento(10) FROM DUAL;
```
##### Opción 2:
- Crear bloque PL/SQL
```sql
DECLARE
	v_total := incremento(v_total);
	dbms_output.put_line('El resultado de sumar 10 es: ' || v_total);
end;
```
