<font style="color:#08F; font-weight:bold">Ver registros de una tabla</font>:
[[08.1 Consultas]]
<font style="color:orange; font-weight:bold">Sintaxis</font>:
```SQL
SELECT * FROM nombre_tabla;
```
<font style="color:#08F; font-weight:bold">Insertar registros</font>:
<font style="color:orange; font-weight:bold">Sintaxis</font> para un registro:
```SQL
INSERT INTO nombre_tabla (columna1, columna2)
VALUES('registro1', 'registro2');
```
<font style="color:orange; font-weight:bold">Sintaxis</font> para varios registros:
```SQL
	INSERT INTO nombre_tabla (columna1, columna2)
	VALUES('registro1', 'registro2'),
	('registro1', 'registro2);
```
- <font style="color:green">Ejemplo</font>:
	```SQL
	INSERT INTO clientes (nombre, codigo_postal)
	VALUES('Martínez e hijos', 03104),
	('Copisteria Pérez', 02104);
	```
<font style="color:#08F; font-weight:bold">Eliminar registros</font> 
- Con sentencia **DELETE**:
	- <font style='color:salmon; font-weight:bold'>Eliminar una fila</font> **sin reiniciar** AUTO-INCREMENT:
		<font style="color:orange; font-weight:bold">Sintaxis</font>:
		```SQL
		DELETE FROM nombre_tabla WHERE condición;
		```
		- <font style="color:green">Ejemplo</font>:
		```SQL
		DELETE FROM clientes WHERE nombre = "Pedro";
		```
	- <font style='color:salmon; font-weight:bold'>Eliminar todas las filas de una tabla</font> **sin reiniciar** AUTO-INCREMENT:
		<font style="color:orange; font-weight:bold">Sintaxis</font>:
		```SQL
		DELETE FROM nombre_tabla;
		```
		- <font style="color:green">Ejemplo</font>:
		```SQL
		DELETE FROM clientes;
		```
- Con sentencia **TRUNCATE**:
	- <font style='color:salmon; font-weight:bold'>Eliminar todas las filas de una tabla</font> **reiniciado** AUTO-INCREMENT:
		<font style="color:orange; font-weight:bold">Sintaxis</font>:
		```SQL
		TRUNCATE TABLE nombre_tabla;
		```
		- <font style="color:green">Ejemplo</font>:
		```SQL
		TRUNCATE TABLE clientes;
		```
<font style="color:#08F; font-weight:bold">Modificar registros</font>:
- <font style='color:salmon; font-weight:bold'>Actualizar una fila</font>:
	- <font style="color:green">Ejemplo</font>:
	```SQL
	UPDATE coche SET año = 2018 WHERE marca = 'Audi';
	```
	- En la **tabla coche**, se actualiza el **año a 2018** de los coches **marca Audi**.
- <font style='color:salmon; font-weight:bold'>Actualizar todas las filas</font>:
	- <font style="color:green">Ejemplo</font>:
	```SQL
	UPDATE coche SET año = 2018;
	```
	- En la **tabla coche**, se actualiza el año a 2018 de todos los coche.
