# LOOP
- Se repite tantas veces como sea necesario hasta que se fuerza su salida con la instrucción EXIT.
<font style="color:salmon; font-weight:bold">Sintaxis:</font>
```sql
DECLARE
-- Sección para declaración de variables
BEGIN
LOOP
	-- Instrucciones
	IF (expresión) THEN
		-- Instrucciones
		EXIT;
	END IF;
END LOOP;
END;
```
# WHILE
- Se repite mientras se cumpla una expresión.
<font style="color:salmon; font-weight:bold">Sintaxis:</font>
```sql
DECLARE
-- Sección para declaración de variables
BEGIN

WHILE (expresión) LOOP
	-- Instrucciones
END LOOP;
END;
```
# FOR
- Se repite tantas veces como se le indique en los identificadores inicio y final.
- En el caso de especificar REVERSE el bucle se recorre en sentido inverso.
<font style="color:salmon; font-weight:bold">Sintaxis:</font>
```sql
FOR contador IN [REVERSE] inicio..final LOOP
	-- Instrucciones
END LOOP;
```
