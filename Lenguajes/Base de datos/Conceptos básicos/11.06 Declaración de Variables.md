<font style="color:salmon; font-weight:bold">Sintaxis:</font>
```sql
nombre_variable [CONSTANT] <tipo_dato> [NOT NULL][:=valor_inicial].
```
- Tipo de dato:
	- Es el tipo de dato que va a poder almacenar la variable.
- Cláusula <font style="color:#08F; font-weight:bold">CONSTANT</font>:
	- Indica la definición de una constante cuyo valor no puede ser modificado.
- Cláusula <font style="color:#08F; font-weight:bold">NOT NULL</font>:
	- Impide que a una variable se le asigne el valor nulo. Por lo que debe inicializarse a un valor diferente de NULL.
<font style="color:green; font-weight:bold">Ejemplo 1:</font>
Declaración de variable v_location, VARCHAR2(15), a la que se le asigna el valor "Granada"
```sql
v_location VARCHAR(15) := 'Granada';
```
<font style="color:green; font-weight:bold">Ejemplo 2:</font>
<font style="font-weight:bold">Procedimiento almacenado aumento de salario.</font> [[11.05 Procedimiento]]
En la empresa donde trabajas se te ha encargado crear un procedimiento almacenado que aumente en un 10% el salario de aquellos empleados cuyo salario esté por debajo de la media.
```sql
CREATE OR REPLACE PROCEDURE Aumento IS salario_med NUMBER;
BEGIN
	SELECT avg(salario) INTO salario_med FROM empleado;
	UPDATE empleado
		SET salario = salario * 1,1
		WHERE salario < salario_med;
END
```
