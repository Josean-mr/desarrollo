## Tablas
- <font style="color:#08F; font-weight:bold">Crear tabla</font>
	 - <font style="color:orange; font-weight:bold">Sintaxis</font>:
	```sql
	CREATE TABLE nombre_tabla;
	```
	- El usuario que crea la tabla se convierte en propietario.
- <font style="color:#08F; font-weight:bold">Ver tablas</font>
	 - <font style="color:orange; font-weight:bold">Sintaxis</font>:
	 ```sql
	SHOW TABLES;
	```
- <font style="color:#08F; font-weight:bold">Eliminar tabla</font>
	- <font style="color:orange; font-weight:bold">Sintaxis</font>:
	```sql
	DROP TABLE nombre_tabla;
	```
## Columnas
- <font style="color:#08F; font-weight:bold">Seleccionar todas las columnas</font>:
	- <font style="color:orange; font-weight:bold">Sintaxis</font>:
		```SQL
		 SELECT * FROM nombre_tabla;
		```
	- <font style="color:green">Ejemplo</font>:
		```SQL
		 SELECT * FROM productos;
		```

- <font style="color:#08F; font-weight:bold">Definir columnas durante la creaci√≥n de una tabla</font>:
	- <font style="color:orange; font-weight:bold">Sintaxis</font>:
		```SQL
		 CREATE TABLE nombre_tabla (
			nombrecolumna     TIPO_COLUMNA [restricciones]
			[,nombrecolumna ......]
			[,restricciones])
		```
		Las **restricciones** establecen reglas de integridad:
		- UNIQUE
		- REFERENCES
		- NULL
		- DEFAULT
		- ...
	- <font style="color:green">Ejemplo</font>:
		```SQL
		CREATE TABLE DEPARTAMENTOS (
			COD INTEGER PRIMARY KEY,
			NOMBRE VARCHAR(45) NOT NULL,
			DIR VARCHAR(45),
			CIUDAD VARCHAR(45),
			TLF VARCHAR(9)
			);
		```
- <font style="color:#08F; font-weight:bold">Agregar columna a tabla ya creada</font>:
	- <font style="color:orange; font-weight:bold">Sintaxis</font>:
		```SQL
		 ALTER TABLE nombre_tabla
		 ADD COLUMN nombre_columna tipo_columna [,RESTRICCIONES];
		```
		- <font style="color:green">Ejemplo</font>:
		```SQL
		 ALTER TABLE pedidos
		 ADD COLUMN producto VARCHAR(50) NOT NULL;
		```
- <font style="color:#08F; font-weight:bold">Cambiar nombre columna</font>:
	- <font style="color:orange; font-weight:bold">Sintaxis</font>:
		```SQL
		 ALTER TABLE nombre_tabla
		 CHANGE nombre_col nuevo_nombre_col TIPO_COLUMNA [,RESTRICC];
		```
		- <font style="color:green">Ejemplo</font>:
		```SQL
		 ALTER TABLE clientes
		 CHANGE dir_cp codigo_postal VARCHAR(5) NOT NULL;
		 ```
- <font style="color:#08F; font-weight:bold">Eliminar columna</font>:
	- <font style="color:orange; font-weight:bold">Sintaxis</font>:
		```SQL
		 ALTER TABLE nombre_tabla
		 DROP COLUMN nombre_columna;
		```
