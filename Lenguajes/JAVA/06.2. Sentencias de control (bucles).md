# ¿Qué son?
Es una característica que facilita la ejecución de un conjunto de instrucciones/funciones repetidamente, mientras que algunas condiciones se evalúen como verdaderas.
# Tipos de bucles
## <font style = "color:FireBrick">while</font>
El bucle while es una sentencia de control de flujo que permite que el código se ejecute repetidamente en función de una condición booleana dada.
El bucle while se puede considerar como una sentencia if repetitiva.

## <font style = "color:FireBrick">do while</font>
El bucle do while es similar al while con la única diferencia qde que comprueba la condición después de ejecutar las instrucciones.

	`sentencia`

## <font style = "color:FireBrick">for</font>
El bucle for proporciona una forma concisa de escribir la estructura de bucle.
La sentencia for consume la inicialización, la condición , y el incremento/decremento en una línea.
**La variable que se crea en el bucle for, solo existirá en el bucle.** Para que exista fuera del bucle, se ha de declarar antes.
### <font style = "color:green">Sintaxis</font>
`bloque 1`

`for (condición de inicialización; condición de prueba; incremento / decremento) {`
	`sentencia`
`}`
