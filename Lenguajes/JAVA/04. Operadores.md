# Tipos de operadores
## <font style="color:blue">Operadores de Asignación.</font>
### Se utilizan para:
- Declaración de datos.
- Para inicializar los datos.
- Para definir el valor inicial:
	- `numero = 10;`
## <font style="color:blue">Operadores Aritmeticos.</font>
### Se utilizan para:
- Manipular datos numéricos.
	- Suma:
		- `resultadoSuma = numero1 + numero2;`
	- Resta:
		- `resultadoResta = numero1 - numero2;`
	- Multiplicación:
		- `resultadoMultiplicación = numero1 * numero2;`
	- División:
		- `resultadoDivision = numero1 / numero2;`
	- Módulo:
		- `resultadoModulo = numero1 % numero2;`
	- Unarios:
		- Incrementar el valor de 1 en 1:
			- `aumentarUno ++;`
		- Decrementar el valor de 1 en 1:
			- `decrementarUno --;`
		- Utilizar la variable e incrementar después:
			- `incrementarDespues ¡++;`
		- Incrementar antes de utilizar la variable:
			- `incrementarAntes ++¡;`
## <font style="color:blue">Operadores Relacionales.</font>
### Se utilizan para:
- Comparar el valor el contenido de una variable respecto a otra.
	- Menor que:
		- <
	- Mayor que:
		- >
	- Igual a:
		- ==
	- No igual a:
		- !=
	- Menor que o igual a:
		- <=
	- Mayor que o igual a:
		- >=
## <font style="color:blue">Operadores Lógicos.</font>
### Se utilizan para:
- Trabajar con valores booleanos.
- Los operadores lógicos son:
	- && // AND el resultado es verdadero si ambas expresiones son verdaderas
	- || // OR el resultado es verdadero si alguna expresión es verdadera
	- ! // NOT el resultado invierte la condición de la expresión
## <font style="color:blue">Operadores Ternario.</font>
### Se utiliza para:
- Versión abreviada de if-else.
- `(5 > 4)?: "Es mayor" : "Es menor";`
