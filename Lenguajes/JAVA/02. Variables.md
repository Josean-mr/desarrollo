# ¿Qué es una variable?
Es el nombre dado a una ubicación en memoria. Es la unidad básica de almacenamiento de un programa.

## <font style="color:orange">Primitivos</font>
### <font style="color:violet">Caracteres</font>
#### Char
- 16 bits
- Representa un carácter Unicode
- Valor por defecto: 'u0000'
- Se declara el valor utilizando comillas simples:
	- `char letraA = 'A';`
### <font style="color:violet">Booleanos</font>
#### boolean
- 1 bit
- True o false
- Valor por defecto:
	- false
### <font style="color:violet">Números enteros</font>
#### byte
- 8 bits
- -128 al 127
- Valor por defecto:
	- 0
#### short
- 16 bits
- -32.768 al 32.767
- Valor por defecto:
	- 0
#### int
- 32 bits
- -2<sup>31</sup> al 2<sup>31</sup>-1
- Valor por defecto:
	- 0
#### long
- 64 bits
- Se ha se agregar una L al final del valor de variable
- -2<sup>63</sup> al 2<sup>63</sup>-1
- Valor por defecto:
	- 0L
### <font style="color:violet">Números enteros o coma flotante</font>
#### float
- 32 bits
- Se ha de agregar una F al final del valor de la variable
- Valor por defecto:
	- 0.0F
#### double
- 64 bits
- Valor por defecto:
	- 0.0d

#### String
- **Son objetos** que pueden representar cadenas de caracteres.
- Valor por defecto:

	- `String cadenaTexto = "Ejemplo de cadena de texto";`

## Declaración
Se puede declarar asignándole un valor:
- `byte edad = 22;`
Se puede declarar sin asignarle valor;
- byte edad;
- Posteriormente se le puede asignar un valor:
	- `edad = 22;`
