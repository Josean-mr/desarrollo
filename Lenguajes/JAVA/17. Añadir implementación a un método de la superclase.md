# Palabra reservada "super"
- Mantener la implementación del método de una superclase y posteriormente ampliar el comportamiento específico de la subclase.
<font style="color:green; font-weight:bold">Ejemplo</font> de la ampliación del comportamiento de la super clase **Window**:
```java
class Ventana extends Window {

	// El parámetro parent es heredado de la super clase Window
	public Ventana (Frame parent) {
	super(parent);
	...
	}
}
```
