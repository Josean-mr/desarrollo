# Creación de cadenas
- Un String o cadena es un array unidimensional cuyos elementos son caracteres.
	- <font style="color:green;font-weight: bold">Ejemplo:</font>
	```java
	public class ArrayDeCaracteres {

		public static void main(String[] args) {

			char cadena[] = new char[4];
			cadena[0] = 'H';
			cadena[1] = 'o';
			cadena[2] = 'l';
			cadena[3] = 'a';
	
			System.out.println(cadena);

		}

	}
	```
- Es más común utilizar la clase String.
-  2 Maneras de crear una cadena mediante String:
	<font style="color:orange;font-weight: bold">Explicitamente:</font>
	```java
	String sMiCadena = new String("Cadena de Texto");
	```
	<font style="color:orange;font-weight: bold">Implicitamente:</font>
	```java
	String sMiCadena = "Cadena de Texto";
	```
	- La forma implícita es más eficiente porque la forma explícita realiza la llamada al constructor 2 veces.

# Creación de cadena vacías
<font style="color:green;font-weight: bold">Ejemplo:</font>
```java
// Forma explícita
String sMiCadena = new String();

// Forma implícita
String sMiCadena = "";
```
- Se puede <font style="color:salmon;font-weight: bold">concatenar cadenas</font> con el operador **+**
	- <font style="color:green;font-weight: bold">Ejemplo:</font>
	```java
	// Declaramos las cadenas
	String miCadena1 = "Hola,";
	String miCadena2 = " ¿qué haces ";

	// Unimos las cadena en una nueva variable
	String cadena1MasCadena2 = miCadena1 + miCadena2 + "por aquí?";

	System.out.println(cadena1MasCadena2);
	```
	- Se imprimirá en consola:
	```console
	Hola, ¿qué haces por aquí?
	```
- También se pueden <font style="color:salmon;font-weight: bold">concatenar con variables de diferente tipo</font>:
	- <font style="color:green;font-weight: bold">Ejemplo:</font>
	```java
	// Declarar variables tipo int
	int num1 = 3;
	int num2 = 8;
	int suma = num1 + num2;

	// Declarar cadena y concatenar con la variable tipo int
	String resultado = "El resultado es " + suma;
	System.out.println(resultado);
	```
	- Se imprimirá en consola:
	```console
	El resultado es 11
	```
# Métodos más usados de la clase String
## lenght()
- Devuelve el tamaño que tiene la cadena.
```java
// Declaración cadena
String saludo = "Hola";

// Almacenar la longitud de la cadena en variable int
int longitud = saludo.length();

System.out.println("La longitud de la cadena es de " + longitud);
```
- Se imprimirá en consola:
```console
La longitud de la cadena es de 4
```
## char charAT(int idex)
- Devuelve el valor indicado como índice.
```java
// Declaración cadena
String saludo = "Hola";

// Almacenar la longitud de la cadena en variable int
int longitud = saludo.length();

// Almacenar el valor del último caracter
char ultimoCaracter = saludo.charAt(longitud - 1);

// Devolver el valor del último caracter
System.out.println("El último caracter es -> " + ultimoCaracter);
```
- Se imprimirá en consola:
```console
El último caracter es -> a
```
## equals
- Compara si las cadenas de texto son iguales o no.
- Se tiene en cuenta mayúsculas y minúsculas.
```java
// Declaración cadena
String saludo = "Hola";

// Almacenar la longitud de la cadena en variable int
int longitud = saludo.length();

System.out.println("La longitud de la cadena es de " + longitud);
```
- <font style="color:green;font-weight: bold">Ejemplo:</font>
```java
// Declaración de cadenas

String compara1 = "A COMPARAR";
String compara2 = "a comparar";

// Variable boolean para comparar si las cadenas son iguales
// Se tiene en cuenta mayúsculas y minúsculas
boolean comparando = compara1.equals(compara2);
System.out.println("Las cadenas son iguales: " + comparando);
```
- Se imprimirá en consola:
```console
Las cadenas son iguales: false
```
- Se puede <font style="color:salmon;font-weight: bold">ignorar la comprobación de mayúsculas y minúsculas</font> mediante los siguientes métodos:
	``` java
	equalsIgnoreCase() // Para no comprobar si son mayusculas o minúsculas
	toUpperCase() // Para convertir los caracteres a mayúsculas
	toLowerCase() // Para convertir los caracteres a minúsculas
	```
- <font style="color:green;font-weight: bold">Ejemplo:</font>
```java
// Declaración de cadenas

String compara1 = "A COMPARAR";
String compara2 = "a comparar";

// Para no tener en cuanta las mayúsculas y minusculas
// Converir a mayúsculas o minúsculas: toUpperCase(), toLowerCase
// o método equalsIgnoreCase() para omitir si el caracter esta en
// mayúsculas o minúsculas
boolean comparando2 = compara1.equalsIgnoreCase(compara2);

System.out.println("Las cadenas son iguales excepto mayúsculas"
	+ " y minúsculas: " + comparando2);
```
- Se imprimirá en consola:
```console
Las cadenas son iguales excepto mayúsculas y minúsculas: true
```
## compareTo
- Se utiliza para comparar las cadena léxicamente.
- Se basa en el valor Unicode los caracteres.
- **Se devuelve un valor menor de 0 si la cadena sobre la que parte es léxicamente menor que la cadena pasada como argumento**.
- Si las dos cadenas son iguales léxicamente se devuelve 0.
- Para omitir la comparación de mayúsculas y minúsculas se puede utilizar <font style="color:salmon;font-weight: bold">compareToIgnoreCase()</font>.
```java
// Declaración de variables
String comparaLexi1 = "Cuervo";
String comparaLexi2 = "cuenco";

// Comparar las variables teniendo en cuenta mayúsculas y minúsculas
int compararLexicamente = comparaLexi1.compareTo(comparaLexi2);

System.out.println("Compración léxica teniendo en cuenta el case: " + compararLexicamente);

// Comparar las variables sin tener en cuenta mayúsculas y minúsculas
int compararLexicamente2 = comparaLexi1.compareToIgnoreCase(comparaLexi2);

System.out.println("Comparación léxica sin tener en cuenta el case: " + compararLexicamente2);
```
- Se imprimirá en consola:
```console
Compración léxica teniendo en cuenta el case: -32
Comparación léxica sin tener en cuenta el case: 4
```
## Búsqueda de caracteres
### int indexOf(int ch)
- Devuelve la posición de la primera ocurrencia de un carácter dentro de la cadena de texto.
- En el caso de que el carácter buscado no exista devolverá -1.
```java
// Declaración de variables
String comparaLexi2 = "cuenco";

// Declaración de variable para guardar la posición
int posicion = comparaLexi2.indexOf('c');

System.out.println("La posición de c de cuenco es: " + posicion);
```
- Se imprimirá en consola:
```console
La posición de c de cuenco es: 0
```
### int indexOf(int ch, int fromIndex)
- Como el anterior método, pero en vez de hacerlo a lo largo de toda la cadena lo hace desde el índice (fromIndex) que se le indique.
```java
// Declaración de variables
String comparaLexi2 = "cuenco";

// Declaración de variable para guardar la posición
int posicion = comparaLexi2.indexOf('c');

System.out.println("La posición de c de cuenco es: " + posicion);

// Buscar el posterior caracter del índice
int posicion2 = comparaLexi2.indexOf('c', posicion + 1);

System.out.println("La posición de la siguiente c de cuenco es: " + posicion2);
```
- Se imprimirá en consola:
```console
La posición de c de cuenco es: 0
La posición de la siguiente c de cuenco es: 4
```
### int lastIndexOF(int ch)
- Índica cuál es la última posición que ocupa un carácter dentro de una cadena.
- Si el carácter no está en la cadena devuelve -1
```java
// Declaración de variables
String comparaLexi2 = "cuenco";

// Buscar la última posicion del caracter
int ultimaPosicion = comparaLexi2.lastIndexOf('c');

System.out.println("La última posición de c en cuenco es: " + ultimaPosicion);
```
- Se imprimirá en consola:
```console
La última posición de c en cuenco es: 4
```
### int lastIndexOf(int, int fromIndex)
- Como el método anterior, pero desde la posición indicada como argumento.
```java
// Declaración de variables
String comparaLexi2 = "cuenco";

// Buscar la última posicion del caracter
int ultimaPosicion = comparaLexi2.lastIndexOf('c');

System.out.println("La última posición de c en cuenco es: " + ultimaPosicion);

// Buscar la anterior posición del caracter del índice
int ultimaPosicion2 = comparaLexi2.lastIndexOf('c', ultimaPosicion - 1);

System.out.println("La posición penúltima de c en cuenco es: " + ultimaPosicion2);
```
- Se imprimirá en consola:
```console
La última posición de c en cuenco es: 4
La posición penúltima de c en cuenco es: 0
```
### String substring(int beginIndex)
- Este método devolverá la cadena que se encuentra entre el índice pasado como argumento (beginIndex) y el final de la cadena origen.
```java
// Declaración de variable
String fraseCadena = "Ciudad Real";

// Buscar cadena posterior al índice
String siguientePosicion = fraseCadena.substring(3);

System.out.println("La última posición de c en cuenco es: " + ultimaPosicion);

System.out.println(siguientePosicion);
```
- Se imprimirá en consola:
```console
dad Real
```
## String substring(int beginIndex, int endIndex)
- Para recuperar la cadena que se encuentra entre dos índices.
```java
// Declaración de variable
String fraseCadena = "Ciudad Real";

// Buscar cadena posterior al índice
String siguientePosicion = fraseCadena.substring(3);

System.out.println("La última posición de c en cuenco es: " + ultimaPosicion);
```

