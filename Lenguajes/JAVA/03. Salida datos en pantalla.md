# Consola
- Se utiliza la clase **System**, el atributo **out** y el método **println()** // ln para realizar un salto de línea. Se puede abreviar con *sout* + tabulación.
	- Ejemplo de imprimir texto literalmente:
		- `System.out.println("Texto a imprimir.");`
	- Ejemplo de imprimir el valor de una variable:
		- `System.out.println(variableTexto);`
	- Ejemplo de imprimir un texto literal y el valor de una variable
		- `System.out.println("Texto a imprimir." + variableTexto);`
