### ¿Qué son?

Son variables que se declaran como una variable normal. Estas variables sirven como características de esta clase. Son las propiedades que tiene.
### Tipos
#### Según tipo de dato
- Datos primitivos:
	- int, double, char...
- Datos complejos:
	- Otros objetos que pertenecen a clases (como String).
#### Según tipo de variable
![[Clases-AtributosSinEjemplos.png]]
- Variable de **instancia**:
	- Son las variables normales.
		- Cada galleta tiene su propia copia.
	- Ejemplo:
		public class Persona {  
			private String nombre; // Atributo de instancia  
			private int edad; // Atributo de instancia
		}
- Variables de **clase** (Static):
	- Implican una sola zona de memoria reservada para las instancias de la clase.
		- Las 3 galletas comparten el valor de la variable.
		- Ejemplo:
			public class Contador {  
				public static int count; // Atributo estático  
			}
- Variables **finales o constantes** (final):
	- Tienen un valor que no cambia a lo largo del programa.
	- Ejemplo:
public class Constantes{  
	public final double PI = 3.14; // Atributo final de instancia  
	public static final int MAX_VALUE = 100; // Atributo final estático  
}
