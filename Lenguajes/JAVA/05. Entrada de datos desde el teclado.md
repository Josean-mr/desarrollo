# <font style = "color:indianred">Scanner</font>
Se trata de una clase incluida en el paquete java.util para obtener la entrada desde diversas fuentes, como la consola, archivos o cadenas de texto. Esta clase permite leer datos de los tipos primitivos como int, double, etc. y también de String.
## Métodos
	nextBoolean() // Leer valores booleanos
	nextByte() // Leer valores byte introducidos por el usuario
	nextDouble() // Leer valores double introducidos por el usuario
	nextFloat() // Leer valores float introducidos por el ususario
	nexInt() // Leer valores int introducidos por el usuario
	nextLine() // Leer valores String introducidos por el usuario
	nextLong() // Leer valores long introducidos por el usuario
	nextShort() // Leer valores short introducidos por el usuario
## Procedimiento
#### Leer valor String
1. Importamos la clase:
	- import java.util.Scanner
2. Creamos instancia de la clase Scanner. Con System.in se indica que leerán datos desde la consola:
	- Scanner lector = new Scanner(System.in);
3.  Creamos variable con el método de lectura de datos:
	- String cadenaTexto = lector.nextLine();

# <font style = "color:indianred">JOptionPane</font>
Es una clase incluida en el paquete javax.swing que provee un conjunto de ventanas diálogo que sirven para proporcionar mensajes al usuario o para solicitar la introducción de algún dato.
Los datos introducidos son de **tipo texto**.
## Métodos
	JOptionPane.showMessageDialog() // Permite mostrar un mensaje
	JOptionPane.showInputDialog() // Permite la entrada de datos
	JOptionPane.ConfirmDialog() // Permite hacer preguntas con varias confirmacones
	JOptionPane.showOptionDialog() // Engloba/unifica los 3 anteriores diálogos
