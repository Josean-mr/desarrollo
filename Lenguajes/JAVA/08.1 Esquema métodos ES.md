```scss

java.io
│
├── File
│   ├── exists()           → Verifica si el archivo existe.
│   ├── getName()          → Obtiene el nombre del archivo.
│   ├── length()           → Devuelve el tamaño en bytes.
│   ├── canRead()          → Verifica si el archivo es legible.
│   ├── getAbsolutePath()  → Obtiene la ruta absoluta del archivo.
│
├── Lectura
│   ├── FileReader (extends Reader)   
│   │   ├── read()         → Lee un solo carácter (devuelve un int).
│   │   ├── read(char[])   → Lee varios caracteres en un array.
│   │   ├── close()        → Cierra el flujo de lectura.
│   │
│   ├── BufferedReader (extends Reader)
│   │   ├── readLine()     → Lee una línea completa como String
							(devuelve null si no hay más líneas).
│   │   ├── read()         → Lee un solo carácter.
│   │   ├── ready()        → Verifica si el flujo está listo para leer.
│   │   ├── close()        → Cierra el `BufferedReader`.
│
├── Escritura
│   ├── FileWriter (extends Writer)
│   │   ├── write(String)  → Escribe una cadena de texto.
│   │   ├── write(char[])  → Escribe un array de caracteres.
│   │   ├── append()       → Agrega texto al final del archivo.
│   │   ├── flush()        → Fuerza la escritura de datos en el archivo.
│   │   ├── close()        → Cierra el flujo de escritura.
│   │
│   ├── BufferedWriter (extends Writer)
│   │   ├── write(String)  → Escribe una línea de texto.
│   │   ├── newLine()      → Escribe un salto de línea (`\n`).
│   │   ├── flush()        → Limpia el buffer y escribe en el archivo.
│   │   ├── close()        → Cierra el `BufferedWriter`.
│
└── java.nio.file
    ├── Files
    │   ├── readAllLines()  → Lee todas las líneas de un
						    archivo en una lista de Strings.
    │   ├── readString()    → Lee todo el contenido del archivo
							en un solo String.
    │   ├── lines()         → Devuelve un Stream de líneas del archivo.
    │   ├── write()         → Escribe una lista de líneas o un
						    String en un archivo.
    │   ├── newBufferedWriter() → Devuelve un `BufferedWriter`
								para escribir en un archivo.
    │
    ├── Paths
    │   ├── get()           → Obtiene la ruta de un
							archivo como objeto Path.
```
