# Estructuras dinámicas
![[73ef7679-8ca8-47f1-bf40-a24e9fb386bf-1.webp]]
1. Se declara una **referencia a un nodo** que será el primero de la lista.
2. Cada nodo de la lista (en la memoria dinámica) contendrá tanto la **propia información del mismo** como una **referencia al nodo siguiente**.
3. Se debe establecer un convenio para identificar el **final de la lista**.
