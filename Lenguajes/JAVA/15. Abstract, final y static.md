# Clases y métodos abstractos
## Clase abstracta
### ¿Qué es?
- **Una clase abstracta es una clase que solo puede tener subclases**.
	- No puede ser ejemplarizada o instanciada.
	- Las subclases son las encargadas de definir los detalles.
	- Para declarar una clase abstracta se utiliza la palabra clave **abstract**.
<font style="color:orange; font-weight:bold">Sintaxis:</font>
```java
abstract class Number {
...
}
```
- La clase abstracta puede contener atributos, métodos, constructores, etc. al igual que una clase convencional. También puede tener **métodos abstractos** (al menos 1).
## Método abstracto
### ¿Qué es?
- Los métodos **no tienen implementación** (sin cuerpo, es decir, sin llaves).
- No tiene cuerpo, llaves "{}".
- Solo existe en una clase abstracta.
- Tiene que ser sobrescrito en una clase hija.
- Termina en punto y coma ";".
<font style="color:orange; font-weight:bold">Sintaxis:</font>
```java
// Con return
abstract modificadorAcceso tipoReturn calcularArea ([param1], [param2]...);

// Sin return
abastrac public void calcularArea(double base, double altura);
```
<font style="color:green; font-weight:bold">Ejemplo:</font>
```java
package abstractos;

// Declaración de clase abstracta
public abstract class Figuras {
	
	// Declaración de atributos
	protected double base;
	protected double altura;

	// Debe haber un método que debe ser abstracto
	public abstract double calcularArea(double base, double altura);
}

// Clase Rectangulo
class Rectangulo extends Figuras {
	
	// Creación de atributo area
	double area;
	
	// Método para calcular el area
	@Override
	public double calcularArea(double base, double altura) {
		this.base = base;
		this.altura = altura;
		this.area = base * altura;
		return area;
	}
	
	// Método toString para imprimir el resultado cuando se le invoque
	@Override
	public String toString() {
		return "El área de un rectángulo con base " + base + " y altura " + altura + " es de " + area;
	}

}

class Triangulo extends Figuras {
	
	// Creación de atributo area
	double area;
	
	// Método para calcular el area
	@Override
	public double calcularArea(double base, double altura) {
		this.base = base;
		this.altura = altura;
		this.area = (base * altura) / 2;
		return area;
	}

	// Método toString para imprimir el resultado cuando se le invoque
	@Override
	public String toString() {
		return "El área de un triángulo con base " + base + " y altura " + altura + " es de " + area;
	}

}

class Main {
	public static void main(String[] args) {
		
		// Creación de objeto de la clase Rectangulo
		Rectangulo rectangulo = new Rectangulo();
		
		// Creación de objeto de la clase Triangulo
		Triangulo triangulo = new Triangulo();
		
		// Invocar método calcularArea con el objeto rectangulo
		rectangulo.calcularArea(10, 5);
		// Invocar método toString para imprimir el resultado
		System.out.println(rectangulo.toString());

		// Invocar método calcularArea con el objeto triangulo
		triangulo.calcularArea(10, 5);
		// Invocar método toString para imprimir el resultado
		System.out.println(triangulo.toString());

	}

}
```

# Clases, métodos y variables finales
## Clase final
### ¿Qué es?
- **Sirve para una clase no pueda tener subclases**.
<font style="color:orange; font-weight:bold">Sintaxis:</font>
```java
final class AlgoritmoAjedrez {
	
	...
	
}
```
## Método final
### ¿Qué es?
- Se utilizan cuando se pretende **proteger los métodos de una clase para que no sean sobrescritos**.
<font style="color:orange; font-weight:bold">Sintaxis:</font>
```java
class AlgoritmoAjedrez {
	...
	final void siguienteMovimiento (Pieza piezaMovida, Posicionen Tablaero nuevaPosicion) {
	}
	...
}
```
# Clases, métodos y variables estáticas
## Clase estática
### ¿Qué es?
- La clase **estática es aquella que no puede ser instanciada** y pertenece a la definición de la clase que la contiene. Es decir, **no es posible crear objetos de una clase estática utilizando el operador "new"**.
- Un <font style="color:green; font-weight:bold">ejemplo</font> de clase estática es la clase **Math**.
```java
package estaticas;

public class EjemploClaseEstaticaMath {

	public static void main(String[] args) {

		double numero = 3.7;
		System.out.println("Numero double: " + numero);

		// Redondear hacia arriba
		double redondeoArriba = Math.ceil(numero);
		System.out.println("Redondeo hacia arriba con método .ceil: " + redondeoArriba);

		// Redondear hacia abajo
		double redondeoAbajo = Math.floor(numero);
		System.out.println("Redondeo hacia abajo con método .floor: " + redondeoAbajo);

		// Valor absoluto
		double valorAbsoluto = Math.abs(numero);
		System.out.println("Valor absoluto con método .abs: " + valorAbsoluto);

		// Generar un número aleatorio entre 0 y 1
		double numeroAleatorio = Math.random();
		System.out.println("Número aleatorio entre 0 y 1 con método .random(): " + numeroAleatorio);

		// Generar un número aleatorio entre 0 y 100 incluidos
		numeroAleatorio = Math.random() * 100;
		System.out.println("Número aleatior entre 0 y 100 con método .random() * 100: " + numeroAleatorio);
		
		// Generar un número aleatorio entre 1 y 100 incluidos
		numeroAleatorio = Math.random() * 99 + 1;
		System.out.println("Número aleatior entre 1 y 100 con método .random() * 99 + 1: " + numeroAleatorio);
	}

}
```

## Variable estática
### ¿Qué es?
- Son útiles cuando se necesita **almacenar información que es compartida por todas las instancias de una clase**, o cuando se **necesita llevar un registro de alguna información en la clase**.
<font style="color:green; font-weight:bold">Ejemplo:</font>
```java
package variable.estatica;

public class Persona {

	/*
	 * contador es una variable estática por lo que tendrá el mismo valor para todas
	 * las instancias de la clase Persona almacena el número de instacias creadas de
	 * la clase Persona. Inicialmente se pone a 0 y se incrementa en 1 cada vez que
	 * se crea una nueva instacia de Persona (cuando se llama a su constructor)
	 */
	private static int contador = 0;
	private String nombre;

	// Constructor de la clase persona
	public Persona(String nombre) {
		this.nombre = nombre;
		contador++; // aumenta en 1 el valor de la variable contador
	}

	// Devueve el valor de la variable estática contador
	public static int getContador() {
		return contador;
	}

	public static void setContador(int contador) {
		Persona.contador = contador;
	}

	public String getNombre() {
		return nombre;
	}

	public void setNombre(String nombre) {
		this.nombre = nombre;
	}

}

class MainPersona {
	public static void main(String[] args) {

		Persona persona1 = new Persona("Juan");
		// Imprimir "Total de personas creadas: 1"
		System.out.println("Total de personas creadas: " + Persona.getContador());

		// Imprimir "Total de personas creadas: 2"
		Persona persona2 = new Persona("Maria");
		System.out.println("Total de personas creadas: " + Persona.getContador());

		// Imprimir "Total de personas creadas: 3"
		Persona persona3 = new Persona("Pedro");
		System.out.println("Total de personas creadas: " + Persona.getContador());

		System.out.println("\n\nEstos son sus nombres..\n");
		System.out.println(persona1.getNombre());
		System.out.println(persona2.getNombre());
		System.out.println(persona3.getNombre());
	}
}
```
