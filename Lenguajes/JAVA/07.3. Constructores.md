 # <font style = "color:orange">¿Qué son?</font>
Es un método ([[07.3_Métodos]]) que se ejecuta al hacer uso de la clase, es decir, al crear un objeto.
# <font style = "color:orange">¿Para que se utilizan los constructores?</font>
Para inicializar el estado del objeto.
# <font style = "color:orange">¿Cuándo se llama al constructor?</font>
Cada vez que se crea un objeto usando la palabra clave <font style = "color:indianred">new()</font>, se invoca al menos un constructor.
