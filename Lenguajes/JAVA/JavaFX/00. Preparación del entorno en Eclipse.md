# 1. Instalar el JDK
- Se debe instalar el Java Development Kit en el sistema.
	- Versiones compatibles con JavaFX: A partir de JDK 8.
# 2. Instalar JavaFX SDK
1. Ingresar en OpenJFX [Link](https://openjfx.io/)
2. Hacer clic en Download.
3. Descargar la versión SDK que corresponda con el sistema.
# 3. Configurar Eclipse
1. <font style="color:#08f; font-weight:bold">Instalar el plugin JavaFX:</font>
	1. Hacemos clic en "help" -> "Eclipse Marketplace".
	![[Pasted image 20250314190348.png]]
	2. En el cuadro de busqueda escribimos FX e instalamos "e(fx)clipse 3.8.0".
	![[Pasted image 20250314190611.png]]
2. <font style="color:#08f; font-weight:bold">Crear librería de usuario:</font>
	1. Ingresamos en el menú: "Windows/Preferences/Java/Build Path/ User libraries".
	![[Pasted image 20250314191123.png]]
	2. Haciendo clic en "New..." escribimos el nombre de la nueva librería de usuario.
	![[Pasted image 20250314191352.png]]
	3. Hacemos clic en "Add External JARs..."
	4. Seleccionamos todas las librerías de la carpeta lib del archivo descargado de OpenJFX (apartado 2). Se recomienda guardar el fichero en un lugar adecuado.
	![[Pasted image 20250314192615.png]]
	5. Se agregarán las librerías.
	![[Pasted image 20250314192704.png]]
	
	6. Aplicamos los cambios y salimos.
# 4. Crear un proyecto JavaFX project
1. <font style="color:#08f; font-weight:bold">Proyecto JavaFX nuevo:</font>
	1. Accedemos a "File/New/Project/JavaFX Project" y hacemos clic en "Next >".
	![[Pasted image 20250314193140.png]]
	2. Escribimos el nombre del proyecto y hacemos clic en "Finish".
	![[Pasted image 20250314193414.png]]
	3. El proyecto aparecerá en el explorador de paquetes.
	![[Pasted image 20250314193527.png]]
2. <font style="color:#08f; font-weight:bold">Agregar librería de usuario:</font>
	1. Clic derecho del ratón sobre el proyecto a acceder a:
		"Buid Path/Add Libraries..."
	![[Pasted image 20250314193909.png]]
	2. Seleccionamos "User Library".
	![[Pasted image 20250314194013.png]]
	3. Seleccionamos la librería creada en el anterior apartado.
	![[Pasted image 20250314201019.png]]
3. <font style="color:#08f; font-weight:bold">Configurar el Build Path:</font>
	1. Clic derecho en el proyecto y escogemos "Path/Configure Build Path..."
	2. ![[Pasted image 20250314201247.png]]
	3. Comprobar que en el "Classpath" de la pestaña "Libraries" está la librería JavaFx. En caso de no estar se debe agregar.
	![[Pasted image 20250315103227.png]]
	4. Si existe algún error, se puede tratar de solucionar eliminando, desde el explorador de paquetes, el archivo "module-info.java" del proyecto.
# 5. Configurar máquina virtual para usar JavaFX
1. Acceder a la web [link](https://openjfx.io/openjfx-docs/) y seleccionar la versión del IDE.
2. Seguimos las instrucciones.
![[Pasted image 20250314202332.png]]
3. Copiamos la instrucción que corresponda con nuesto sistema operativo.
-  Windows:
```
--module-path "\path\to\javafx-sdk-23.0.2\lib" --add-modules javafx.controls,javafx.fxml
```
- Linux/Mac:
```
--module-path /path/to/javafx-sdk-23.0.2/lib --add-modules javafx.controls,javafx.fxml
```
3. Clic derecho en el proyecto "Run As/Run Configurations..."
![[Pasted image 20250314202914.png]]
4. Ponemos el proyecto y la clase correspondientes:
![[Pasted image 20250314203127.png]]
5. En la pestaña "Arguments" pegamos la instrucción copiada.
![[Pasted image 20250314203412.png]]
6. Cambiamos el path por el de la carpeta lib de la librería descargada.
![[Pasted image 20250314203623.png]]
![[Pasted image 20250314203642.png]]
7. Hacer clic en "Apply" y en "Run". Si todo ha salido bien, aparecerá un cuadro sin contenido.
![[Pasted image 20250315102707.png]]
8. En caso de que exista un error, se puede probar a eliminar el archivo "module-info.java" de la carpeta "src".
