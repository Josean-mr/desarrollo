# ¿Qué es?
- Es la capacidad de un objeto de una clase de comportarse de diferentes maneras o asumir diferentes formas en función del contexto.
# Mecanismos para lograr el polimorfismo en Java
- <font style="color:#08f; font-weight:bold">Polimorfismo de subtipos</font>
- <font style="color:#08f; font-weight:bold">Sobrecarga de métodos</font>
## <font style="color:#08f; font-weight:bold">Polimorfismo de subtipos</font>
- Proporciona la capacidad de redefinir por completo el método de una superclase en una subclase.
<font style="color:green; font-weight:bold">Ejemplo:</font>
```java
package polimorfismo.subtipos;

public class Animal {
	
	void hacerSonido() {
		System.out.println("Este animal hace un sonido.");
	}
}

class Gato extends Animal{
	
	@Override
	void hacerSonido() {
		System.out.println("El gato maulla.");
	}
}

class Perro extends Animal{
	
	@Override
	void hacerSonido() {
		System.out.println("El perro ladra.");
	}
}

class Main {
	public static void main(String[] args) {
		
		// Crear objeto de la clase Animal
		Animal miMascota;
		
		// Crear instancia de la clase Animal
		miMascota = new Animal();
		// Llamar al método de la clase Animal
		miMascota.hacerSonido();
		
		// ------ Polimorfismo de subtipos --------
		
		// Instancia de objeto de la subclase Gato
		miMascota = new Gato();
		// Llamar al método de la subclase Gato
		miMascota.hacerSonido();
		
		// ---------------------------------------
		
		// Instancia de objeto de la subclase Perro
		miMascota = new Perro();
		// Llamar al método de la subclase Gato
		miMascota.hacerSonido();
		
	}
}
```

## <font style="color:#08f; font-weight:bold">Sobrecarga de métodos</font>
- Es la capacidad de definir múltiples métodos en una misma clase con mismo nombre, pero con diferentes parámetros.
<font style="color:green; font-weight:bold">Ejemplo:</font>
```java
package polimorfismo.sobrecargametodos;

// Clase calculadora con métodos de suma
public class Calculadora {

	// Método que retorna el resultado de la suma de enteros
	int sumar(int a, int b) {
		return a + b;
	}

	// Método que retona el resultado de la suma de doubles con el mismo nombre
	double sumar(double a, double b) {
		return a + b;
	}
}

class Main {
	public static void main(String[] args) {

		// Crear objeto de clase Calculadora
		Calculadora calculadora;
		// Crear instancia de clase
		calculadora = new Calculadora(); Calculadora

		/*
		* Llamada al método suma que corresponda el
		* tipo de variable con el tipo de return
		*/
		int sumaEnteros = calculadora.sumar(34, 54);
		double sumaDoubles = calculadora.sumar(32.4, 21.1);

		System.out.println("Resultado suma de enteros: " + sumaEnteros);
		System.out.println("Resultado suma de doubles: " + sumaDoubles);
	}
}
```
