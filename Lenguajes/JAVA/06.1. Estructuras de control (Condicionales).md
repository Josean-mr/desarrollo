# ¿Qué son?
Las instrucciones de control alternativas o condicionales permiten determinar el flujo de ejecución de un programa evaluando una condición lógica.
# Tipos de estructuras de control
## <font style = "color:FireBrick">if</font>
- Evalúa una condición lógica, si se cumple, se ejecutará el bloque de instrucciones.
### <font style = "color:green">Sintaxis</font>
`bloque 1`
`if (condicion){`
	`bloque 2`
`}`
`bloque 3`
![[Pasted image 20240825184745.png]]
## <font style = "color:FireBrick">if - else</font>
- La instrucción if se puede complementar con else para definir un bloque de instrucción que se ejecutarán en caso de que la condición evaluada no se cumpla.
### <font style = "color:green">Sintaxis</font>
`bloque 1`
`if (condicion){`
	`bloque 2`
`}else{`
	`bloque 3`
`}`
`bloque 4`
![[Pasted image 20240825195531.png]]
### if - else anidados
`bloque 1`
`if (condicion){`
	`bloque 2`
`}else if (condicion){`

## <font style = "color:FireBrick">switch</font>
- La instrucción tendrá ciertos casos que se ejecutarán dependiendo del valor evaluado.
### <font style = "color:green">Sintaxis</font>
`bloque 1`
`switch (valor) {`
	`case valor1: bloque valor1;`
	`break;`
	`case valor2: bloque valor2;`
	`break;`
	`default: bloque def;`
	`break;`
`}`
`bloque 2`
