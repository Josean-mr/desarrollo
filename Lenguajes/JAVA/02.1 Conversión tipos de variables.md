# <font style = "color:orange">String a primitivos</font>
## A tipos enteros y decimales
## <font style = "color:green">Método "parse"</font>
`// 1. Se declara variable de tipo cadena`
`String cadenaNumero = "12";`

`// 2. Se convierte a tipo int utilizando la clase Integer`
`int numeroConvertido = Integer.parseInt(cadenaNumero);`

## <font style = "color:green">Método "valueOf"</font>
`// 1. Se declara variable de tipo cadena`
`String cadenaNumero = "12";`

`// 2. Se convierte a tipo int utilizando la clase Integer`
`int numeroConvertido = Integer.valueOf(cadenaNumero);`

## A tipo character
## <font style = "color:green">Método "charAt"</font>
`// 1. Se declara variable de tipo cadena`
`String cadenaTexto = "Hola mundo";`

`// 2. Se convierte a char. El número entre paréntesis representa la posición en el objeto String`
`char letraSegunda = cadenaTexto.charAt(1);`
# <font style = "color:orange">Primitivos a String</font>
## A tipos enteros y decimales
## <font style = "color:green">Método "toString"</font>
`// 1. Se declara variable de tipo entero o decimal`
`float numeroFlotante = 1.12423F;`

`// 2. Se convierte a tipo String utilizando la clase de origen (Float)`
`String numeroCadena = Float.toString(numeroFlotante);`
## <font style = "color:green">Método "valueOf"</font>
`// 1. Se declara variable de tipo entero o decimal`
long numeroLong = 112342L;

// 2. Se convierte a tipo String utilizando la clase de origen (Long)
String numeroCadena = Long.toString(numeroLong);
## <font style = "color:green">Con variable + dobles comillas</font>
`// 1. Se declara variable`
`float numeroFloat = 1.12342F;`

`// 2. Se convierte a tipo String concatenando la variable con las dobles comillas`
`String numeroCadena = numeroFloat + """;`
# <font style = "color:orange">Entre primitivos</font>
Las conversiones pueden ser **implícitas** o **explícitas**.
- <font style = "color:green">Implícitas</font>:
	- Si la variable destino es de mayor capacidad que la variable destino se puede realizar una conversión **automática**.
	- Ejemplo:
		`short numeroShort = 12;`
		`int numeroInt = numeroShort;`
- <font style = "color:green">Explícitas</font>:
	- Cuando la variable destino es de menor capacidad se ha de realizar la conversión de manera **manual**.
	- Para realizar la conversión explícita, se ha de escribir entre paréntesis el tipo de variable de origen.
	- Ejemplo:
		`int numeroInt = 23;`
		`short numeroShort = (int) numeroInt;`
	- Hay que considerar si el valor cabe en el rango de la variable destino.
