# ¿Qué es?
PHP es un lenguaje de programación que se ejecuta en el lado del servidor.
Su código se puede integrar con el de HTML para crear webs dinámicas.
Los documentos PHP tienen extensión .php.
PHP se puede insertar en el código de un documento web mediante el uso de etiquetas:
```php
<?php ?>
```

# Instalación de servidor en Linux
## Instalar servidor web XAMPP
1. Ir a la web del XAMPP y decargar el instalador:
	1. [Xampp]([XAMPP Installers and Downloads for Apache Friends](https://www.apachefriends.org/es/index.html))
2. Iniciar XAMPP:
	1. sudo /opt/lampp/lampp start
3. Comprobar la instalación:
	1. En navegador: localhost
## Uso de Visual Studio Code con XAMPP
- Ruta de los archivos del servidor XAMPP:
	- /opt/lammp/htdocs/
- Generar los archivos del servidor:
	- sudo touch /opt/lammp/htdocs/
- Cambiar el propietario de los archivos al usuario actual para trabajar con Visual Studio Code:
	- a. `sudo chown $USER:root /opt/lampp/htdocs/*.*`
	- b. También se puede modificar el propietario de la carpeta /opt/lampp/htdocs para poder crear archivos sin sudo (archivos serán del usuario):
		- sudo chown $USER:root /opt/lampp/htdocs
	- c. Una practica recomendable es la de cambiar el directorio del servidor para utilizar la carpeta del usuario:
		- Abrir archivo de configuración:
			- sudo nano /opt/lampp/etc/httpd.conf
		- Modificar las líneas:
			- `DocumentRoot`
			- `<Directory>`
		- Importante. Para que el daemon del servidor web pueda acceder a la ubicación. La ruta debe tener permisos de ejecucion para otros usuarios.
			- `chmod o+x /home/<usuario>`
