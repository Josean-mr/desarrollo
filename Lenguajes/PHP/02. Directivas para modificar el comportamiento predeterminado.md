Las directivas de PHP se pueden configurar desde el archivo <font style="color:#0088ff; font-weight:bold">php.ini</font>, <font style="color:#0088ff; font-weight:bold">htaccess</font> en apache y desde el propio código de PHP.
<font style="color:green; font-weight:bold">Ejemplo:</font>
- En PHP:
```php
ini_set('memory_limit', '512M'); // Cambia el límite de memoria a 512 MB
```
- En php.ini:
```
memory_limit = 512
```
