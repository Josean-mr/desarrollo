Se puede utilizar tanto <font style="color:#0088ff; font-weight:bold">echo</font> como <font style="color:#0088ff; font-weight:bold">print</font> para imprimir mensajes en php:
Ejemplo:
```php
<?php echo "mensaje a mostrar" ?>
```
El mensaje a imprimir a de estar entre comillas simples ('mensaje') o dobles ("mensaje").
También permite mostrar comillas:
```php
<?php echo 'Las "comillas dobles" se mostrarán' ?>
```
# Concatenar
Se pueden concatenar mensajes con el . (punto).
```php
<?php echo "El principo de un texto" . " y el final" ?>
```
