# Linux basado en Debian
## Directorio de instalación
```bash
/etc/apache2
```
## DocumentRoot
- Donde se alojan los documentos de la aplicación, del usuario, etc., que se servirán a los navegadores.
```bash
/var/www
```
### Server Root
- Donde se encuentran los ficheros de configuración del servidor.
-  Por seguridad se encuentra en otra ubicación.
- Este es el mismo directorio que el instalación.
```bash
/etc/apache2
```
- En esta ubicación se encuentra el archivo de configuración <font style="color:#08f; font-weight:bold">apache2.conf</font> (en otras distribuciónes se pude llamar httpd.conf).
