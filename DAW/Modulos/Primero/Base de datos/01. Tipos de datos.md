# 2. Tipos de datos
<font style="color:salmon; font-weight:bold">01. Números exactos</font>:
- <font style="color:#08f; font-weight:bold">INT</font> o <font style="color:#08f; font-weight:bold">INTEGER:</font>
	- 32 bits.
- <font style="color:#08f; font-weight:bold">SMALLINT</font>:
	- 16 bits.
<font style="color:salmon; font-weight:bold">02.  Números decimales</font>:
- <font style="color:#08f; font-weight:bold">NUMERIC</font>:
	- Precisión, escala.
	- Tiene una longitud fija (parte entera más decimal)
- <font style="color:#08f; font-weight:bold">DECIMAL</font>:
	- Precisión, escala.
	- No tiene una longitud mínima.
- Se deben especificar dos datos:
	- **Precisión**: Número de dígitos en el número.
	- **Escala**: Cantidad de dígitos a la derecha del punto decimal.
- escala <= precisión.
<font style="color:salmon; font-weight:bold">03.  Números aproximados</font>:
- <font style="color:#08f; font-weight:bold">REAL</font>:
	- Números reales.
	- Hasta 7 dígitos de mantisa.
	- 4 bytes.
- <font style="color:#08f; font-weight:bold">FLOAT</font>:
	- Números flotantes.
	- Necesita especificar la precisión del número.
	- 8 Bytes.
	- 15 dígitos de precisión.
<font style="color:salmon; font-weight:bold">04. Cadena de caracteres</font>:
- <font style="color:#08f; font-weight:bold">CHAR</font> (número) o <font style="color:#08f; font-weight:bold">CHARACTER</font> (número):
	- Caracteres fijos.
	- Siempre se almacena el número de caracteres almacenados en número.
- <font style="color:#08f; font-weight:bold">VARCHAR</font> (número):
	- Caracteres variables.
	- Se almacenan como máximo el número de caracteres dado por número.
<font style="color:salmon; font-weight:bold">05. Cadena de bits</font>:
- <font style="color:#08f; font-weight:bold">BIT</font> (número):
	- Se suele utilizar para almacenar flags para el control de algún proceso.
	- En número se almacena la cantidad de bits.
<font style="color:salmon; font-weight:bold">06. Datos de fechas y horas</font>:
- <font style="color:#08f; font-weight:bold">DATE</font>:
	- Define datos en formato **fecha**.
-  <font style="color:#08f; font-weight:bold">TIME</font>:
	- Define datos en formato **hora**.
-  <font style="color:#08f; font-weight:bold">DATETIME</font> o <font style="color:#08f; font-weight:bold">TIMESTAMP</font>:
	- Unión de un tipo DATE y de un tipo TIME.
